# Rick & Morty Explorer

Aplicación web full-stack que consume la API pública de Rick and Morty.

## Stack

- **Frontend:** Angular 19 (standalone components, signals)
- **Backend:** Node.js + Express (proxy API)
- **Package manager:** pnpm

## Estructura

```
NYXN_Full/
├── frontend/   # Angular app
└── backend/    # Node.js Express API
```

## Desarrollo

### Backend
```bash
cd backend
pnpm install
pnpm dev
```

### Frontend
```bash
cd frontend
pnpm install
pnpm start
```
